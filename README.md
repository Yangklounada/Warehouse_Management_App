# Система управления рабочими пространствами и заказами

> Десктопное приложение, разработанное на **C# (.NET Framework, Windows Forms)** с использованием **SQLite** в качестве локальной базы данных. Приложение предназначено для управления рабочими пространствами, товарами и заказами, а также поддерживает регистрацию пользователей и генерацию отчетов.

---

# Описание проекта

Данный проект представляет собой информационную систему для организации рабочего процесса внутри небольшого предприятия или магазина.

Приложение позволяет пользователям создавать собственные рабочие пространства, вести список товаров, оформлять заказы, хранить информацию в локальной базе данных и формировать отчеты.

В качестве пользовательского интерфейса используется технология **Windows Forms**, а все данные сохраняются в базе **SQLite**, благодаря чему приложение не требует установки отдельного сервера базы данных.

---

# Основные возможности

## Авторизация пользователей

При запуске приложения пользователю доступно окно входа.

Реализованы возможности:

- регистрация нового пользователя;
- вход в систему;
- хранение учетных записей в SQLite;
- проверка корректности введенных данных.

После успешной авторизации открывается основное окно приложения.

---

## Рабочие пространства

После входа пользователь может создавать отдельные рабочие пространства.

Для каждого рабочего пространства доступны:

- собственный список товаров;
- собственные заказы;
- независимое хранение данных;
- возможность открытия и редактирования.

Каждое рабочее пространство принадлежит конкретному пользователю.

---

## Управление товарами

В приложении реализовано управление ассортиментом товаров.

Для каждого товара хранится информация:

- название;
- цена;
- количество.

Поддерживаются операции:

- добавление товара;
- удаление товара;
- просмотр списка товаров;
- сохранение изменений в базе данных.

---

## Создание заказов

Пользователь может создавать новые заказы.

Во время оформления заказа доступны:

- выбор товаров;
- указание количества;
- автоматический расчет общей стоимости;
- сохранение заказа в базе данных.

Заказ формируется на основании товаров, зарегистрированных в рабочем пространстве.

---

## Работа с базой данных

Все данные приложения сохраняются в локальной базе SQLite.

Хранятся следующие сущности:

- пользователи;
- рабочие пространства;
- товары;
- заказы;
- позиции заказа.

Использование SQLite позволяет запускать приложение без дополнительной настройки сервера базы данных.

---

## Генерация отчетов

В приложении предусмотрено отдельное окно формирования отчетов.

Отчеты позволяют получить информацию о:

- созданных заказах;
- товарах;
- общей стоимости заказов.

---

# Архитектура проекта

Проект построен по принципу разделения пользовательского интерфейса и логики работы с данными.

Основные формы приложения:

- WelcomeWindow — окно авторизации;
- RegistrationWindow — регистрация пользователя;
- App — главное окно приложения;
- WorkSpace — рабочее пространство пользователя;
- Product — создание нового товара;
- NewOrderFrame — оформление заказа;
- ReportFrame — просмотр отчетов;
- NameWorkSpace — создание нового рабочего пространства.

Все данные сохраняются посредством SQLite.

## WelcomeWindow

- переход к регистрации;
- проверка учетных данных.

- отображение рабочих пространств;

- навигацию по приложению.

---

## WorkSpace

Основное рабочее окно.

Позволяет:

- работать со списком товаров;
- просматривать заказы;
- создавать новые записи;
- взаимодействовать с базой данных.

---

## Product

Форма создания нового товара.

Позволяет указать:

- название товара;
- стоимость;
- количество.

---

## NewOrderFrame

Форма оформления заказа.

Выполняет:

- выбор товаров;
- расчет общей стоимости;
- сохранение заказа.

## ReportFrame

Используется для формирования отчетов по данным приложения.

# Работа с базой данных

В качестве СУБД используется SQLite.

Основные преимущества выбранного решения:

- отсутствие необходимости установки сервера;
- хранение данных в одном файле;
- простое резервное копирование;
- высокая скорость работы для настольного приложения.

Для доступа к данным используется библиотека:

```
System.Data.SQLite
```
